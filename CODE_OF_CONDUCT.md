# Code of Conduct

## Our Standards

Participants are expected to communicate respectfully, stay focused on technical quality, and avoid harassment, personal attacks, or discriminatory language.

## Scope

This code of conduct applies to issues, pull requests, discussions, and other project spaces related to BondLens AI.

## Enforcement

The repository owner may remove comments, close issues, or block participants who violate these expectations.

## Project-Specific Boundary

This project discusses financial data for learning and engineering demonstration. Do not use project spaces to solicit or provide investment advice, trading recommendations, return guarantees, or personalized financial decisions.
